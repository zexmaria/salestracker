Bem vindo

SalesTracker é um sistem para gerenciamento de vendas.Você pode cadastrar clientes, vendedores, produtos e vendas, além de gerar relatórios detalhados das suas transações. Desenvolvido como parte de um projeto pessoal, este sistema utiliza tecnologias como Django e Django REST Framework (DRF).
